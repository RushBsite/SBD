# SBD
## **프로젝트 소개**

공동 배달 주문 서비스

## **프로젝트 목표**

인접 지역 / 같은 가게에서 음식배달을 시켜먹는 사람들을 모아 배달비를 줄일 수 있게 하는 서비스

## **지원 기능**

게시판 형식 / 시간제한 플랫폼

## **사용 스택**

React, Nodejs, mariaDB, k8s, aws, 위치기반 Openapi

## **repository 규칙**
|폴더|설명|
|-|-|
`front`|  React 작업물
`NodeJS`|  B.E 작업물
`src`|  기타 필요 외부 소스 저장 _(ex)img)_
